# 台股 AI 投資評價模型

> AI-powered Taiwan stock valuation model using Claude — 7-module scoring system covering financials, growth, valuation, Capex, and Taiwan-specific indicators. Outputs structured investment reports in minutes.
>
> 台股 AI 投資評價模型｜結合 Claude AI 與手動財務數據，7 模組量化評分（財務健康／成長性／估值／Capex／台灣特殊指標），5 分鐘產出完整評價報告，內建半導體次產業專屬評分機制。

**版本：v1.1**　｜　**最後更新：2026-07-05**

## 這個專案能做什麼

你填入約 5 項財務數字，Claude AI 自動完成以下工作：

1. **Phase 0｜產業偵測**：自動判斷個股是否屬於台灣半導體供應鏈，並細分為晶圓代工／IC設計／封測／設備材料／記憶體等次產業，決定採用一般或半導體專用評分門檻。
2. **Phase 1｜讀取資料**：讀取你填好的手動表單，確認必填欄位齊全。
3. **Phase 2｜網路搜尋**：自動補充法說會展望、外資籌碼、月營收趨勢等資訊。
4. **Phase 3｜7 模組評分**：套用固定評分邏輯（一般產業或半導體次產業專屬門檻）。
5. **Phase 4｜產出報告**：生成 Markdown 報告，報告開頭先顯示總分與評語（結論先講重點）。
6. **Phase 4.5｜自動驗證**：存檔前執行 8 項品質檢查，攔截常見錯誤。

**輸出格式：** Markdown 報告 / 標準 HTML / 新手友善版 HTML（皆可直接在 Cowork 對話框下指令產出）

每個數字都標記來源（✅ 手動輸入 / 🔍 網路搜尋 / 🤖 AI 估算），方便人工覆核。

## 快速上手（初次設定）

### Step 1｜安裝 Claude Desktop
下載並安裝：[claude.ai/download](https://claude.ai/download)，登入帳號。

### Step 2｜下載本專案
```bash
git clone https://github.com/herecomesmickey/taiwan-stock-ai-valuation.git
```
或點右上角 **Code → Download ZIP**，解壓縮到任意位置。

### Step 3｜建立 Cowork Project
1. 打開 Claude Desktop → 點左上角「**Cowork**」分頁
2. 點「**+ 新增 Project**」，命名為「台股評價模型」
3. 點「**選擇資料夾**」，選取 `台股評價模型/` 資料夾
4. 點「**Project 指示**」，將 `INSTRUCTIONS.md` 的**全部內容貼入**
5. 儲存 → 設定完成 ✅

> 此設定只需做一次，之後每次進入 Project 即可直接分析。

## 每次使用流程

### Step 1｜複製表單（30 秒）
把 `manual_data/_TEMPLATE.md` 複製並重新命名：
```
2330_manual.md   ← 台積電
2454_manual.md   ← 聯發科
```
可參考 `manual_data/2330_manual.md` 了解填寫格式。

### Step 2｜填入 5 項必填數據（5～8 分鐘）

同時開啟三個網頁(可依據個人習慣與偏好自行使用其他資料來源)：

| 網頁 | 網址 | 取得資料 |
|------|------|---------|
| 財報狗 | `statementdog.com/analysis/{代號}` | EPS、毛利率、本益比河流圖 |
| 玩股網 | `wantgoo.com/stock/{代號}` | ROE、負債比率、月營收 |
| 公開資訊觀測站 | `mops.twse.com.tw` | 現金流量表（Capex、OCF） |

五項必填：EPS 與 3 年 CAGR、獲利能力五指標＋FCF、P/E 歷史三區間、Capex 與折舊、近 6 個月月營收。

選填：外資投信籌碼、法說展望、庫藏股、地緣政治風險；半導體股另可填良率趨勢與產能利用率。

### Step 3｜讓 AI 分析（約 1 分鐘）
在 Cowork 對話框輸入：
```
分析 2330
```
AI 完成後，報告先在對話視窗顯示供確認，再存入 `reports/` 資料夾。

### Step 4｜（選用）產出美化 HTML 報告

報告存檔後，直接在 Cowork 對話框輸入：

```
產出HTML報告 2330
```

產出新手友善版（含白話解釋財務術語）：

```
產出新手版HTML報告 2330
```

AI 會讀取對應的 Markdown 報告，生成美化 HTML 並存入 `reports/` 資料夾，用瀏覽器開啟即可閱讀或列印。

## 7 模組評分說明（v1.1）

### 模組 0｜產業偵測（不計分）
分析前自動判斷個股是否屬於台灣半導體供應鏈，並細分次產業：

| 次產業 | 評分模式 |
|--------|---------|
| 晶圓代工 / IC 設計 | 半導體專用門檻（毛利率優門檻 45%） |
| 封測（OSAT） | 半導體專用門檻（毛利率優門檻 22%） |
| 半導體設備材料 | 依公司自身 5 年歷史區間相對判斷 |
| 記憶體 | 依公司自身 5 年歷史區間相對判斷 |
| 其他 / 一般產業 | 一般門檻 |

偵測結果與依據寫入報告「資料來源說明」區塊。

---

### 各模組分數

| 模組 | 滿分 | 評估內容 |
|------|------|---------|
| 模組 1 | 不計分 | 基本資料、供應鏈位置、Revenue Breakdown（選填） |
| 模組 2 | 25 分 | 財務健康：毛利率、營業利益率、淨利率、ROE、負債比率、FCF 品質 |
| 模組 3 | 20 分 | 成長性：營收 YoY、EPS 3年CAGR、月營收趨勢、法說展望 |
| 模組 4 | 20 分 | 估值：P/E 河流圖位置、同業比較、合理股價空間；另附歷史估值價格對照區間（不計分） |
| 模組 5 | 15 分 | Capex 資本支出：Capex／折舊比、Capex YoY、FCF margin、法說一致性；IC設計改評研發費用率 |
| 模組 6 | 20～24 分 | 台灣特殊指標：月營收動能、外資籌碼、投信籌碼、法說態度、庫藏股、地緣政治風險；製造業另加良率＋產能利用率（+4分） |
| 模組 7 | — | 綜合評分與 200 字投資評語 |

> **半導體模式說明：** 模組 2、3、5 採次產業專屬門檻；模組 6 良率與產能利用率加分項僅適用擁有自有產能的次產業（晶圓代工／封測／記憶體），IC 設計（Fabless）不適用。

---

### 評分結果判定

**一般產業（滿分 100 分）**
- 🟢 70 分以上：值得積極關注
- 🟡 50～69 分：持續觀察追蹤
- 🔴 50 分以下：暫時迴避

**製造／半導體（滿分 104 分，適用良率＋產能利用率加分項時）**
- 🟢 73 分以上：值得積極關注
- 🟡 52～72 分：持續觀察追蹤
- 🔴 52 分以下：暫時迴避

## 報告格式

每份報告包含以下區塊（由上至下）：

```
📌 結論（先講重點）         ← 開頭直接顯示總分與 200 字評語
資料來源說明               ← 含產業偵測結果
模組 1 基本資料
模組 2 財務健康　[XX/25]
模組 3 成長性　[XX/20]
模組 4 估值　[XX/20]
模組 5 Capex資本支出　[XX/15]
模組 6 台灣特殊指標　[XX/20 或 XX/24]
模組 7 綜合評分
AI 投資評語（200字）
✅ 驗證清單（8 項）
⚠️ 免責聲明
建議下次更新日期
```

### 8 項存檔前驗證清單

| # | 驗證項目 |
|---|---------|
| 1 | 財報期間一致性（所有財務數字來自同一期間） |
| 2 | 股價時效性（使用當日或前一交易日收盤價） |
| 3 | EPS 計算基礎已標明（TTM / FY / 預估） |
| 4 | 殖利率計算正確（現金股利 ÷ 當前股價） |
| 5 | 月營收 YoY 基期正確（對比去年同月） |
| 6 | AI 估算欄位有依據（每項附估算說明） |
| 7 | 評分加總一致（各模組加總 = 模組 7 總分） |
| 8 | **估值價格帶換算一致性（EPS 基礎與 P/E 判定相同，無操作性字眼）** |

## 常見問題

**Q：要付費才能用 Claude？**
A：Claude 有免費方案，Cowork 功能包含在內。付費方案可使用更多訊息次數與更強的模型。

**Q：每次都要重新設定嗎？**
A：不用。Cowork Project 設定一次後，資料夾連結和 AI 指示都會保留。

**Q：可以同時分析多支股票嗎？**
A：可以。分別填好各股票的 manual.md，再依序輸入「分析 2330」「分析 2454」即可。

**Q：半導體股和一般股的分析有什麼差異？**
A：AI 會在 Phase 0 自動偵測並套用對應門檻，不需要手動切換。偵測結果會寫入報告中。

**Q：模組 4 的「歷史估值價格對照區間」是進場建議嗎？**
A：不是。這是依歷史 P/E 區間換算的價格對照，**僅供參考，不構成進場或操作建議**。
